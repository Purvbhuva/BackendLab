# Task A2: Basic Calculator

A functional calculator application built with NextJS and React client components.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division, modulo
- **Advanced Functions**: Clear (AC), backspace, negate (±), decimal support
- **Real-time Display**: Shows current calculation and operations
- **State Management**: Uses React hooks to manage calculator state
- **Responsive Design**: Beautiful gradient UI with smooth interactions

## Installation

```bash
npm install
```

## Running the Application

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Operations Supported

- **+** : Addition
- **−** : Subtraction
- **×** : Multiplication
- **÷** : Division
- **%** : Modulo (Remainder)
- **AC** : All Clear
- **⌫** : Backspace
- **±** : Negate
- **.** : Decimal

## Key Features

- Chain calculations
- Decimal number support
- Immediate feedback on operations
- Clean, intuitive interface
