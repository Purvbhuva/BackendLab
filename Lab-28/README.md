# Lab 28: NextJS Server Actions Demo

This project demonstrates various implementations of Server Actions in Next.js with TypeScript:

## Features

1. **Demo A**: Inline server action that prints form data
2. **Demo B**: Server action defined in a separate file
3. **Demo C**: Server action with form validation

## Tech Stack

- Next.js 14 with App Router
- TypeScript 5
- React 18
- Server Actions

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Server Actions Demonstrated

- Inline server actions in components
- Reusable server actions in separate files with TypeScript interfaces
- Form data validation in server actions
- Server-side form handling with type safety
