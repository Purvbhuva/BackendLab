Completed tasks:
1) Student controller with helloWorld method
   - File: src/student.controller.ts
   - Route: GET /students/hello
   - Response: hello world

2) Student controller with findAll, findOne, insert, update and delete methods
   - GET /students -> findAll()
   - GET /students/:id -> findOne(id)
   - POST /students -> insert(body)
   - PUT /students/:id -> update(id, body)
   - DELETE /students/:id -> delete(id)

3) Three different controllers with all above methods
   - StudentController: /students
   - TeacherController: /teachers
   - CourseController: /courses

Each controller includes:

Sample endpoint tests (PowerShell):

- Invoke-RestMethod http://localhost:3000/students/1

- Invoke-RestMethod -Method PUT http://localhost:3000/students/1 -Body '{"name":"Updated Ali"}' -ContentType 'application/json'
